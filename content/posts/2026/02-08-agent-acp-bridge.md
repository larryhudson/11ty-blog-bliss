A side project I've been chewing on. The idea: you shouldn't have to sit in a terminal babysitting an agent.

`agent-acp-bridge` lets you talk to Claude Code (and other agents that speak the ACP protocol) from inside the tools you already work in — a Linear comment, a Slack thread, a GitHub PR. You kick the agent off from where you are, it does the work, and it reports back in the same place, so you don't have to keep switching context to check on it.

Code is on [GitHub](https://github.com/larryhudson/agent-acp-bridge).
